# 监控所持有基金的净值，收盘后根据昨日净值和持仓计算今日收益并邮件/微信提醒
* 收盘后根据昨日净值和份额计算盈亏，其余时间休眠不占用资源
* 自选邮件/微信提醒方式
* 灵活的配置，自由确定基金代码和份额，即可自动监控计算
* 基金净值超过期望自动提醒卖出。低于期望提醒买入
